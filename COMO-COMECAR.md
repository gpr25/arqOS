# Como Começar — ArqOS
# m2 Intelligence

---

## Antes de instalar

Você precisa de dois itens:

**1. Claude Code** — o terminal de IA da Anthropic
```
npm install -g @anthropic-ai/claude-code
```
Ou acesse claude.ai/code para instruções completas.

**2. Uma conta Claude** com plano Pro ou acima

Sem os dois, o sistema não funciona. Se ainda não tem, configure agora antes de continuar.

---

## O que você está instalando

O ArqOS é o sistema operacional do seu negócio.

Não é um chatbot. Não é um template. É um sistema construído com o contexto real da sua empresa — que fica mais inteligente a cada uso.

Ao final do onboarding, você terá:

- **14 agentes especializados** com conhecimento de nível sênior em cada área — comercial, marketing, atendimento, conteúdo, dados, lançamento, documentos, financeiro e mais
- **21 comandos operacionais** — do script de atendimento à proposta comercial, do carrossel ao relatório de performance
- **Memória operacional** — o sistema registra contexto, decisões e aprendizados. Cada sessão o torna mais preciso.
- **Skills de publicação** — Instagram, LinkedIn e integrações externas

---

## Instalação — 2 passos

**1. Clone o repositório**

Abra o terminal e execute:

```bash
git clone https://github.com/m2intelligence/arqos.git
```

A pasta `arqos/` será criada no diretório atual. Ela vai crescer aqui — escolha um local permanente antes de clonar (Documentos, Área de Trabalho, onde preferir).

**2. Inicie o ArqOS**

```bash
cd arqos
claude
```

Quando o Claude Code abrir, digite:

```
/instalar-self
```

O onboarding começa.

---

## Atualizações futuras

Para receber novos agentes, comandos e melhorias, rode dentro da pasta `arqos/`:

```bash
git pull
```

Os arquivos do seu negócio (criados pelo onboarding) não são afetados.

## Como o onboarding funciona

**Tempo estimado:** 20 a 35 minutos.

Antes das perguntas, o sistema passa por cinco etapas:

1. **Pede seu nome e o da empresa** — para personalizar tudo desde a primeira mensagem.

2. **Verifica as ferramentas instaladas** — Node.js, Python, Playwright e extensões úteis. Se não tiver algo, você decide se quer instalar (o sistema guia) ou seguir sem.

3. **Lê os materiais que você mandar** — apresentação, manual de marca, portfólio, qualquer documento interno. Usa como contexto e pula as perguntas que já estão respondidas.

4. **Analisa seu site e redes sociais** — acessa com Playwright (ou método alternativo) e mapeia posicionamento, tom de voz, produtos e público.

5. **Seleciona o perfil** — Incorporadora, Imobiliária, Corretor ou Outro negócio.

Depois, 3 a 4 blocos de perguntas adaptadas ao perfil. Uma pergunta de cada vez. Ao final de cada bloco, os arquivos são criados imediatamente — e o sistema sintetiza o que capturou antes de avançar.

## Antes de começar

Tenha à mão qualquer material sobre o negócio:

- Apresentação da empresa
- Manual de marca ou guia de identidade visual
- Portfólio ou tabela de produtos
- Scripts de atendimento, processos documentados

Quanto mais contexto o sistema tiver antes de perguntar, mais preciso o resultado — e menos perguntas você responde.

---

## Se o seu negócio não é imobiliário

O ArqOS foi construído com foco no mercado imobiliário, mas opera com qualquer tipo de empresa.

Ao selecionar "Outro tipo de negócio", o sistema adapta a estrutura ao seu modelo. Os agentes têm exemplos imobiliários que você pode ajustar — e cada um indica exatamente o que adaptar para o seu setor.
