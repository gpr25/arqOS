# /mapear-rotinas — Mapear rotinas e identificar automações

Entrevista sobre o que você faz repetidamente toda semana e transforma isso em fichas de automação priorizadas.

Se $ARGUMENTS for fornecido, foque nas rotinas da área mencionada (ex: marketing, comercial, operações, atendimento).

## Entrevista de rotinas

Uma pergunta por vez, espere a resposta antes de avançar:

1. Pensa na sua semana típica. Quais são as 3 tarefas que você mais repete?
2. Para cada uma: quanto tempo gasta? É manual ou tem parte automatizada?
3. Qual dessas tarefas seria mais valioso automatizar primeiro?
4. Existe alguma tarefa que você evita fazer porque é chata — mas que impacta o negócio?
5. Tem alguma tarefa que só você sabe fazer e que travaria a operação se você sumisse?

## Após a entrevista

Para cada rotina identificada, gerar uma ficha:

```
ROTINA: [nome]
Frequência: [diária / semanal / por demanda]
Tempo atual: [estimativa]
Potencial de automação: [alto / médio / baixo]
Como automatizar: [descrição do skill ou fluxo]
Próximo passo: [ação concreta]
```

Ao final, propor uma ordem de prioridade e perguntar por qual quer começar.
