# /gerar-conteudo — Criar conteúdo calibrado com a identidade da empresa

Você vai criar uma peça de conteúdo calibrada com o contexto real desta empresa.

Leia antes de começar:
- `branding/brand.md` — tom de voz, posicionamento, frases proibidas
- `marketing/estrategia.md` — pilares de conteúdo, canais, público

Se $ARGUMENTS for fornecido, use como tema, formato ou canal de partida.

## Como executar

**1. Confirmar o briefing**

Pergunte (em uma mensagem só, não uma por vez):

```
Para criar o conteúdo, preciso confirmar:

1. Formato: Reel, carrossel, post estático, legenda ou roteiro?
2. Canal: Instagram, LinkedIn, WhatsApp ou outro?
3. Objetivo: educar, converter, autoridade ou engajamento?
4. Tema ou ideia central (pode ser uma frase, um problema, um insight):
5. Tem alguma restrição ou referência que devo considerar?
```

Se $ARGUMENTS já trouxer essas informações, não pergunte o que já está claro.

**2. Criar o conteúdo**

Use o agente de conteúdo como referência de estrutura e checklist.
Produza a peça completa — não um esqueleto, não um rascunho.

**3. Apresentar com checklist de qualidade**

Ao final de cada peça, apresentar:

```
CHECKLIST DE QUALIDADE:
✓ Gancho funciona sozinho
✓ Específico para este negócio e público
✓ Tom alinhado com a marca
✓ CTA único e claro
[x] [item que não foi possível verificar ou que precisa de ajuste]

SUGESTÃO DE ADAPTAÇÃO:
→ Para [outro canal]: [como adaptar em uma linha]
```

**4. Registrar**

Ao final, perguntar se o conteúdo foi aprovado e se deve ser registrado em `memory/campanhas.md`.
