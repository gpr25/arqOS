# /publicar-tema — Pacote completo de conteúdo a partir de um tema

Orquestre um pacote completo de conteúdo a partir de um único tema.

Leia antes de começar:
- `branding/brand.md`
- `marketing/estrategia.md`

Tema: $ARGUMENTS

Se nenhum tema for fornecido, pergunte qual tema quer trabalhar e qual o objetivo (educar, posicionar, converter).

## Entregáveis do pacote

Gere em sequência — um por vez, aguardando aprovação antes de avançar:

### 1. Artigo de blog
- Título SEO-otimizado
- Estrutura com H2s e H3s
- Introdução (problema que o leitor reconhece)
- Desenvolvimento (3-5 seções com profundidade real)
- Conclusão com CTA para contato ou diagnóstico
- Meta description (150-160 caracteres)
- Mínimo 1.200 palavras

### 2. Carrossel Instagram/LinkedIn
- 5-7 slides com copy completo
- Adapta os pontos principais do artigo
- Formato visual alinhado com `branding/identidade.md`

### 3. Legenda para publicação
- Versão Instagram (até 2.200 caracteres)
- Versão LinkedIn (até 3.000 caracteres, mais técnica)
- Hashtags relevantes para cada plataforma

### 4. Sugestão de agendamento
- Melhor dia e horário para publicar (com base em `marketing/calendario.md` se existir)
- Sequência sugerida: LinkedIn primeiro ou Instagram?

Após gerar tudo, perguntar se quer salvar o conteúdo em `memory/campanhas.md`.
