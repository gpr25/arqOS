# /instalar-self — Instalar ArqOS Self System

Você vai guiar o onboarding autônomo do ArqOS Self System.

Leia antes de começar:
- `onboarding/INSTALAR.md` — sistema completo de entrevista adaptativa

Se $ARGUMENTS for fornecido, use como nome ou contexto inicial da empresa.

## Como executar

Siga o `onboarding/INSTALAR.md` na íntegra, respeitando as regras:

1. **Um bloco por vez** — nunca apresente todas as perguntas de uma vez
2. **Aceitar materiais** — se o usuário disser que tem um arquivo, pare e espere
3. **Criar arquivos ao final de cada bloco** — não espere o onboarding terminar
4. **Sinalizar lacunas** — campos sem resposta ficam como `[PREENCHER]`
5. **Nunca conteúdo genérico** — tudo baseado nas respostas reais do usuário

Comece pela mensagem de abertura do `INSTALAR.md` e siga o fluxo até a finalização completa.
